# قائمة ما قبل الرفع للإنتاج

- [ ] ضبط `APP_TIMEZONE=Asia/Riyadh` في `.env`
- [ ] ضبط `APP_ENV=production`
- [ ] ضبط `APP_DEBUG=false`
- [ ] إضافة cron job لتشغيل السكجولر:
  ```
  * * * * * cd /path-to-project && php artisan schedule:run >> /dev/null 2>&1
  ```
  (بدون هالسطر، الراتب ما بينزل تلقائياً ولا السكجولر يشتغل)
